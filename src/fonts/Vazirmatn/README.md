# Vazirmatn Font

Add font files to `static/` folder. Expected structure:

```
Vazirmatn/
└── static/
    ├── Vazirmatn-Light.ttf
    ├── Vazirmatn-Regular.ttf
    ├── Vazirmatn-Medium.ttf
    ├── Vazirmatn-SemiBold.ttf
    └── Vazirmatn-Bold.ttf
```

Or variable font: `Vazirmatn-VariableFont_wght.ttf` at root.

Download from [Vazirmatn GitHub](https://github.com/rastikerdar/vazirmatn/releases).
