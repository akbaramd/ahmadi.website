# Poppins Font

Add font files to `static/` folder. Expected structure:

```
Poppins/
└── static/
    ├── Poppins-Light.ttf
    ├── Poppins-Regular.ttf
    ├── Poppins-Medium.ttf
    ├── Poppins-SemiBold.ttf
    └── Poppins-Bold.ttf
```

Or variable font: `Poppins-VariableFont_wght.ttf` at root.

Download from [Google Fonts](https://fonts.google.com/specimen/Poppins).
